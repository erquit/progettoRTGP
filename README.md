# progettoRTGP
My project for the Real Time Graphics Programming at University of Milan during my Masters in Computer Science (LM-18)

Note: In this version the library "libassimp.a" normally found in "Libraries/lib" was removed since it was too heavy to load on github, this library is required for the correct execution of the project, for user who are looking to run this project it's advised to follow the "Building Assimp" section, found on the LearnOpenGL site: https://learnopengl.com/Model-Loading/Assimp
